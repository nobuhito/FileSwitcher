# USAGE

```
FileSwitcher init
FileSwitcher list
FileSwitcher set hogehoge

FileSwitcher init -t /path/to/targetfile
FileSwitcher list -t /path/to/targetfile
FileSwitcher set -t /path/to/targetfile

or

FileSwitcher init -c /path/to/configfile
FileSwitcher list -c /path/to/configfile
FileSwitcher set hogehoge -c /path/to/configfile
```
